Explicación de Sub-Lenguajes SQUL-LMD, LDD, LCD

Estimado (a) estudiante, le damos una grata la bienvenida al Objeto Virtual de Información (OVI), donde se abordaran los temas propuestos para el curso administración de base de datos, donde le daremos una breve introducción y las generalidades, esperamos que el curso y los temas contenidos en este sean de su agrado. También podremos a su disposición referencias para que el tema sea entendible y ameno.

En las sesiones se encuentran contenidos donde le explicaran una breve introducción de cada concepto de los lenguajes y sus sentencias SQL, al mismo tiempo la administración de una base de datos en el gestor de base de datos SQL Server, por medio de las sentencias LCD, LDD y LMD y la administración de seguridad.

SQL (DDL, DML, DCL Y TCL): DDL (Data Definition Language): Permite crear y modificar la estructura de una base de datos. DML (Data Manipulation Language): Permite recuperar, almacenar, modificar, eliminar, insertar y actualizar datos de una base de datos. DCL (Data Control Language): Permite crear roles, permisos e integridad referencial, así como el control al acceso a la base de datos. TCL (Transactional Control Language): Permite administrar diferentes transacciones que ocurren dentro de una base de datos.
